void MioPlayer.play ( ) throws JavaLayerException [inline]

Fa partire il brano.

Gestisce sia il caso in cui il brano non sia mai andato prima in esecuzione, sia che il brano sia stato ripreso dopo una pausa. Qui vi è la creazione e il lancio del thread. Scatena una JavaLayerException. 

  {
    synchronized (lock) 
    {
      switch (stato) 
      {
        case NONINIZIATO:
          final Thread t = new Thread(){ public void run() { riproduci();}};
          stato = INESECUZIONE;
          t.start();
          break;
        case INPAUSA:
          riprendi();
          break;
        default:
          break;
      }
    }
  }
void MioPlayer.riprendi ( ) [inline]

Riprende l'esecuzione dalla pausa.

Cambia lo stato della riproduzione e notifica il thread che dovrebbe essere in stato di attesa, con notifyAll lo "risveglia". 

  {
    synchronized (lock) 
    {
      if (stato == INPAUSA)
      {
        stato = INESECUZIONE;
        lock.notifyAll();
      }
    }
  }
void MioPlayer.riproduci ( ) [inline, private]

Riproduce effettivamente il brano.

Funzione che finchè vi sono frame da eseguire, le esegue. Se lo stato è INPAUSA il thread viene "addormentato" con wait() e verrà "risvegliato" solo con un notify() o un notifyAll()

Se non vi sono più frame da eseguire (quando player.play(1) ritorna false), viene invocato il metodo close(); 

  {
    while (stato != FINITO) 
    {
      try {
        if (!player.play(1)) 
          break;
      } catch (final JavaLayerException e) {break;}
      
      
      synchronized (lock) 
      {
        while (stato == INPAUSA) 
        {
          try 
          {
            lock.wait();
          } catch (final InterruptedException e) 
          {
            break;
          }
        }
      }
    }
    close();
  }
