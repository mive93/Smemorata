Classe SalvaFileChooser.

Questa classe estende JFileChooser e lo specializza per il salvataggio di una LibreriaMultimediale sul file scelto attraverso il FileChooser.

Se il file selezionato è già esistente si apre una finestra di dialogo per confermare la sovrascrittura o annullarla.
